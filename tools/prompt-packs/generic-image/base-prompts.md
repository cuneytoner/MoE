# Base Generic Image Prompts

## A. Clean Realistic Object Image

Prompt:

```text
realistic product-style image of [SUBJECT], clean background, natural lighting, detailed material texture, practical real-world proportions, high clarity
```

## B. Concept Reference Image

Prompt:

```text
concept reference image of [SUBJECT], realistic materials, clear silhouette, practical design, balanced composition, natural daylight
```

## C. Mood Board Style Image

Prompt:

```text
visual mood board image for [SUBJECT], material references, color palette, texture details, practical design cues, clean layout
```

## D. Marketing Visual

Prompt:

```text
clean marketing visual of [SUBJECT], realistic scene, natural lighting, simple composition, high clarity, no text, no logo
```

## E. Before/After Idea Visual

Prompt:

```text
realistic before and after concept image for [SUBJECT], renovation idea, clear visual contrast, practical materials, natural lighting
```

## Notes

- Replace `[SUBJECT]`.
- Avoid asking for exact text inside the image.
- Avoid relying on generated dimensions.
