# Technical Visual Reference Prompts

## A. Connection Visual Reference

Prompt:

```text
realistic close-up visual reference of [CONNECTION_TYPE], practical materials, visible fasteners, clean lighting, no labels, no text, for visual intent only
```

## B. Assembly Sequence Visual

Prompt:

```text
realistic assembly step visual for [ASSEMBLY], one clear construction step, practical tools, visible parts, clean work area, no text, no labels
```

## C. Material Detail Visual

Prompt:

```text
realistic material detail visual of [MATERIAL_OR_PART], close-up, texture clarity, practical real-world use, natural lighting
```

## D. Safety Reference Visual

Prompt:

```text
realistic safety reference visual for [TASK], protective gear, clear work area, practical tool handling, no text, no labels
```

## E. Non-Engineering Technical Concept

Prompt:

```text
technical-looking visual reference of [SUBJECT], clean schematic composition, simple parts, no dimensions, no labels, visual intent only
```
