# Technical Reference Prompt Pack

## Purpose

This pack contains prompts for technical visual reference images. Use it to explore the look of connections, assemblies, materials, safety scenes, and technical-looking concepts.

## Technical Visual Reference Prompts

Technical reference prompts can help clarify visual intent for:

- connection details
- assembly steps
- material close-ups
- safety scenes
- non-engineering technical concepts

## Difference From Deterministic Drawings

Deterministic drawings are generated from explicit geometry, dimensions, and known rules. AI-generated technical visuals are approximate images and may invent parts, omit details, or show impossible geometry.

Use deterministic drawing tools for measured plans, elevations, labels, dimensions, and construction review.

## Warning

Do not use AI-generated technical visuals as engineering truth.
