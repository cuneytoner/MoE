# Architecture Prompt Pack

## Purpose

This pack contains reusable prompts for architecture and spatial concept images. Use it for exterior concepts, interior concepts, outdoor structures, and material or lighting variants.

## Exterior Concepts

Exterior prompts are for visualizing building massing, facades, entries, garden structures, roof coverage, rain protection, and sun shading.

## Interior Concepts

Interior prompts are for room layout ideas, workshop organization, renovation concepts, storage, lighting, and material palettes.

## Outdoor Structures

Outdoor structure prompts cover garden studios, pergolas, shade structures, covered entries, small shelters, and other practical exterior additions.

## Material And Lighting Variants

Use material and lighting variants to compare visual direction: wood, metal, wall finishes, roof surfaces, daylight, overcast scenes, shaded areas, and task lighting.

## Safety Warning

Architecture images are visual references only, not construction documents. Do not use them as proof of dimensions, code compliance, drainage, structural integrity, or build sequencing.
