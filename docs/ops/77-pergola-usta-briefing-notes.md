# 77 Pergola Usta Briefing Notes

These notes explain how to use selected generated images when discussing the pergola project with a carpenter/usta.

## Main Rule

Images are visual intent references.

They are not engineering drawings, static calculations, connection plans, or validated construction details.

## How To Use The References

- Use R1/R2 to explain the general look.
- Use R3/R4 to explain rain protection and covered walkway.
- Use R5/R6 only to explain "I want visible connection details", not actual bracket design.
- Bring real measurements separately.
- Bring material list separately.
- Bring connection/delikt plan separately.
- Do not ask the usta to copy AI bracket details blindly.

## Real Project Reminders

- total wall-side width around `5.1 m`
- depth around `1.9 m`
- `10x10 cm` posts
- `5x10 cm` beams/rafters
- fully covered roof
- translucent/polycarbonate roof is acceptable
- right-side door canopy/extension
- left side half-height protection option

## Questions For The Usta

- Where should posts be placed for the real wall-side width?
- What roof slope and drainage detail is practical?
- What post base detail should be used on the existing floor?
- Which brackets, screws, through bolts, or washers are appropriate?
- How should the right-side door canopy/extension connect to the main frame?
- Is the left side half-height protection practical with the available lumber?

## What Not To Ask

- Do not ask the usta to copy AI-generated bracket geometry.
- Do not ask the usta to infer exact measurements from the images.
- Do not ask the usta to build from visual references alone.
