# 106 Generic Media Generation Roadmap

This roadmap defines planned generic image capabilities beyond the pergola case study.

## Planned Generic Image Capabilities

- generic prompt packs
- style presets
- model/runtime selection later
- single image run
- controlled prompt variants
- output naming
- reference-board selection
- compare outputs
- dashboard cards
- prompt metadata capture

Generic prompt packs are now introduced in M34.1.

## Planned Categories

- architecture exterior
- architecture interior
- garden / outdoor structures
- product mockups
- technical visual references
- marketing visuals
- before/after renovation ideas

## Output Review

Each generated output should keep:

- prompt text
- workflow/model context
- output path
- filename
- file size
- generated date/time
- category
- review notes
- reference-board decision

## Safety

- real generation remains operator-controlled
- no automatic overnight generation
- no uncontrolled batch
- generated images stay under runtime
- no image binaries in Git

## Next Work

Start with a generic prompt pack structure, then connect output review to dashboard cards and reference-board metadata.
