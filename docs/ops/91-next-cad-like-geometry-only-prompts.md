# 91 Next CAD-like Geometry-only Prompts

This page defines the next generation prompts for more CAD-like diagrams.

They do not run generation.

After testing geometry-only prompts, deterministic SVG drawing is now the preferred direction.

## G1 Side Elevation Geometry Only

```text
flat orthographic 2D CAD-style line diagram of a lean-to pergola side elevation, house wall vertical line on left, front post vertical line on right, sloped roof line from high rear to lower front, simple rafters as parallel lines, no text, no numbers, no labels, no perspective, no 3D, no shading, white background, black lines only
```

## G2 Top Plan Geometry Only

```text
flat top-down orthographic 2D CAD-style line diagram of a rectangular wall-side pergola plan, long wall line, shallow depth, square post markers, parallel rafter lines, small right-side door canopy rectangle, roof overhang outline, no text, no numbers, no labels, no perspective, white background, black lines only
```

## G3 Beam-Post Connection Geometry Only

```text
flat 2D CAD-style line diagram of a wooden pergola beam-to-post connection, vertical post rectangle, horizontal beam rectangle, two L angle bracket shapes, four bolt circles with washer rings, no text, no numbers, no labels, no perspective, no wood texture, white background, black lines only
```

## G4 Roof Panel Screw Geometry Only

```text
flat top-down 2D CAD-style line diagram of polycarbonate roof panel on parallel rafters, panel outline, overlap edge, drip edge, washer screw circles, no text, no numbers, no labels, no perspective, white background, black lines only
```
