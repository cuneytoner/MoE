# 65 Pergola Prompt Lessons Learned

This note summarizes lessons from the improved pergola prompt run.

## Project-Like Prompt Worked Best

The project-like covered pergola prompt was the strongest direction.

It produced a practical lean-to style structure near a wall, with natural pine and covered translucent roof panels.

## Covered Translucent Roof Language Helps

Phrases such as:

- `fully covered translucent roof panels`
- `covered roof protecting the walkway`
- `translucent polycarbonate roof sheets`

help steer the output away from open decorative pergolas.

## Rain Prompt Improved With Material And Surface Detail

The rain prompt improved when it included:

- wet floor
- damp wood
- visible rain streaks
- cloudy overcast sky
- realistic moisture and reflections

This was stronger than simply adding `rainy weather`.

## Technical Prompt Still Needs More Detail

The technical output improved but remained too polished.

"construction documentation photo" alone is not enough.

Future technical prompts should include:

- close-up
- visible screw heads
- metal angle brackets
- washers
- bolt holes
- pencil marks
- tape measure
- labeled beams
- practical site mess
- unfinished wood

## Practical Lesson

For technical outputs, describe the exact connection or construction detail, not just the overall pergola.

Good next targets:

- beam-to-post connection
- post base
- rafter and roof screw detail
- wall ledger connection
- brackets and fasteners
