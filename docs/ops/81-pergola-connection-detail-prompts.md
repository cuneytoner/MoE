# 81 Pergola Connection Detail Prompts

These prompts target drawing-like connection and component details.

They do not run generation.

Connection prompts need simplified schematic framing. Review [85-technical-drawing-run-result-review.md](85-technical-drawing-run-result-review.md) through [87-next-simplified-technical-drawing-prompts.md](87-next-simplified-technical-drawing-prompts.md).

## C1 Beam-Post Connection

```text
technical close-up line drawing of wooden pergola beam-to-post connection, 10x10 cm pine post, 5x10 cm beam sitting on or bolted to post, galvanized L angle brackets, through bolts, washers, screw heads, hole positions marked, black line drawing with labels
```

## C2 Post Base

```text
technical detail drawing of 10x10 cm wooden pergola post base on concrete or stone paving, galvanized post base shoe, anchor bolts, washers, drainage gap under wood, labels, practical construction detail, black line drawing
```

## C3 Roof Sheet Fastening

```text
technical detail drawing of translucent polycarbonate roof sheet fastened to 5x10 cm wooden rafters, screws with rubber washers, overlap direction, drip edge, waterproofing note, labels, clean black line drawing
```

## C4 Side Half-Height Panel

```text
technical detail drawing of left side half-height protection panel for pergola, 70 to 90 cm high, simple pine frame, vertical slats or flat panel, attached between posts, labels, black line drawing
```

## C5 Door Canopy Extension

```text
technical drawing of right-side small door canopy extension connected visually to main pergola, pine posts, 5x10 rafters, translucent roof panel, rain protection over doorway, labels, black line drawing
```
