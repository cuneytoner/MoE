# 80 Pergola Measured View Prompts

These prompts target measured drawing-like views for the real pergola project.

They do not run generation.

## V1 Front Elevation

```text
clean carpenter technical drawing, front elevation of a DIY wooden pergola, total width 5.10 meters, pine timber, 10x10 cm posts, 5x10 cm beams, fully covered translucent roof, simple measured labels, black line drawing on white background, practical construction drawing, not artistic render
```

## V2 Side Elevation

```text
clean carpenter technical drawing, side elevation of a lean-to wooden pergola beside house wall, depth 1.90 meters plus 30 cm roof overhang, rear high side near wall, front lower side, 10x10 cm posts, 5x10 cm rafters, translucent covered roof, dimension labels, black line drawing, practical usta plan
```

## V3 Top Plan View

```text
top-down plan drawing of a wall-side DIY wooden pergola, total wall-side width 5.10 meters, depth 1.90 meters, right-side door canopy extension, post positions shown as 10x10 cm squares, rafters shown, roof overhang shown, dimension labels, clean black technical drawing
```

## V4 Combined Usta Sheet

```text
single-page carpenter plan sheet for a DIY wooden pergola, includes front elevation, side elevation, top plan view, simple dimension labels, 10x10 posts, 5x10 beams, translucent roof panels, practical Turkish backyard construction, black line drawing on white background
```
