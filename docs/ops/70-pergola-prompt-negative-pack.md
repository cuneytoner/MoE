# 70 Pergola Prompt Negative Pack

This page collects negative and avoid phrases for project-specific pergola runs.

Workflow support for a separate negative prompt may need verification before use.

If the workflow does not support a negative prompt field, incorporate avoid phrases into the main prompt with `not ...` wording.

## Candidate Avoid Phrases

- luxury resort
- fantasy architecture
- open roof
- impossible structure
- oversized palace beams
- glossy showroom render
- no visible construction details
- decorative only
- tropical villa
- huge garden pavilion
- metal-only structure
- concrete pergola
- unrealistic floating beams

## Main-Prompt Avoid Wording

Use short avoid phrases at the end of the main prompt when needed:

```text
not luxury, not resort style, not decorative only, not open roof, not fantasy architecture, not glossy showroom render
```

For technical prompts:

```text
not luxury render, not decorative only, not metal-only structure, not concrete pergola, not unrealistic floating beams
```
