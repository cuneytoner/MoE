# Output Card Metadata Detail Review Template

Use this template when reviewing output card metadata details.

## Review

- Date/time:
- Dashboard URL:
- Card tested:
- Card type:
- metadata_available true?:
- Metadata API status:
- Prompt visible for image?:
- Geometry visible for drawing?:
- Safety label visible?:
- Script visible?:
- Raw JSON fallback safe?:
- No edit button?:
- No rerun/generate button?:
- No delete/move/rename?:
- Any arbitrary path exposed?:
- Issues found:
- Git safety result:

## Notes

- Metadata detail view should fetch by `card_id`.
- Metadata should be displayed as text only.
- Sidecar JSON should remain runtime-only.
