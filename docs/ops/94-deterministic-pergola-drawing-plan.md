# 94 Deterministic Pergola Drawing Plan

This plan documents the decision to move real pergola technical drawings to deterministic code-generated geometry.

It does not create drawing files yet.

M33.5 creates the first SVG drawing tool skeleton.

## Why Deterministic Drawing Is Needed

Flux image generation is useful for visual intent, but it is not reliable for:

- exact dimensions
- true top-down plans
- orthographic elevations
- joinery details
- bracket geometry
- screw/bolt placement
- repeatable CAD-style output

The real pergola needs drawings where dimensions, labels, lines, and geometry come from code and reviewed assumptions.

## Why SVG First

SVG is a good first target because:

- it is text-based
- it is deterministic
- it can be generated with standard Python
- it is easy to inspect in a browser
- it can hold lines, labels, dimensions, and simple shapes
- it can later be exported to PDF or converted to other formats

## Why PDF/DXF Later

PDF and DXF should come later because:

- PDF export needs layout and print decisions
- DXF needs CAD compatibility decisions
- the first milestone should prove the geometry model and labels
- premature export formats can add complexity before the drawing content is stable

## What Must Be Generated By Code

- side elevation
- top plan
- front elevation
- beam-post detail
- roof sheet layout
- deterministic labels
- dimension arrows and dimension text
- simple post, beam, rafter, bracket, bolt, and washer placeholders

## What Must Remain Manually Reviewed

- real site dimensions
- post positions
- roof slope
- drainage direction
- material choices
- cut list
- bracket types
- screw and bolt sizes
- anchor choices
- structural sanity
- weatherproofing details

## Safety

- Generated drawings should go under runtime by default.
- Source code can live in the repo.
- Generated SVG/PDF/DXF outputs should not be committed unless explicitly intended as small documentation assets.
- PDF/DXF should not be committed by default.
- No destructive cleanup.
- Do not use AI-generated dimensions or bracket geometry as source of truth.
