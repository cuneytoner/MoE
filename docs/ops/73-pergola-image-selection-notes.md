# 73 Pergola Image Selection Notes

This note prepares for selecting useful generated images as references.

Generated outputs are visual references only. They are not engineering drawings, static calculations, construction plans, or validated joinery details.

The first reference board is documented in [75-pergola-image-reference-board.md](75-pergola-image-reference-board.md).

## Useful References From The First Project-Specific Run

- P1 is useful as a project overview reference.
- P4 is useful as a rain-protection reference.
- T1 is useful as a technical/detail direction reference.

## Selection Rules

- Use generated images to communicate intent.
- Do not treat AI-generated joinery as structurally valid.
- Do not use generated images to replace real static/detail plans.
- Do not infer exact dimensions from generated images.
- Do not commit generated image binaries to Git.
- Record selected output paths and filenames in notes.

## What To Look For

For project overview:

- narrow wall-side walkway feel
- covered translucent roof
- visible pine timber frame
- practical backyard construction

For rain protection:

- wet walkway
- damp wood
- visible rain mood
- roof coverage over the walking path

For technical/detail direction:

- close-up framing
- visible bolts, screws, washers, and brackets
- useful wood grain and pencil-mark cues
- plausible construction language, even if the generated joinery itself is not trusted

## What To Reject

- luxury resort styling
- open decorative roof
- huge pavilion scale
- impossible floating beams
- structurally confusing bracket placement
- images that hide the timber frame
- images that imply exact construction plans
