# Environment Profiles

This directory contains source-controlled examples and templates for assigning MoE / AI-Brain-OS roles to machines.

Allowed files in this directory:

- `*.example.yaml`
- `*.template.yaml`

Do not commit real local profile files such as `pc1.local.yaml`, secrets, generated runtime state, logs, pid files, model files, or machine-specific credentials.

Runtime data belongs under `/home/cuneyt/MoE/runtime`.

Model files belong under `/home/cuneyt/MoE_Models_Backup`.

The source code lives under `/home/cuneyt/DiskD/Projects/MoE/codebase`.
